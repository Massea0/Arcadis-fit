### Installation

1. **Clone the repository**
```bash
git clone <repository-url>
cd arcadis-fit
```

2. **Backend Setup**
```bash
cd backend
npm install
cp .env.example .env
# Configure your environment variables
npm run dev
```

3. **iOS Setup**
```bash
cd ios
pod install
open ArcadisFit.xcworkspace
```

4. **Android Setup**
```bash
cd android
./gradlew build
```
